Please start all Scripts and executables with the command line format
shown in the .sh files of the containing directories.
After the execution command the input strings are folowed preceeded by obligatory 
parameters only if listed in the .sh file. More Informations in the other Readme files.

All input strings has a length >= 2 and are at least 2 !
The number AND length of input strings in the OpenCL Versions must be a power of 2
for performance reason ! ( 2,4,8,16,...  e.g. 4 x 8-character strings)
